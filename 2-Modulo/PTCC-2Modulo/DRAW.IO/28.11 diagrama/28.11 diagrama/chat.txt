# 🟥 **5. Emergencia (AQUI É O MAIOR PROBLEMA!)**

Nesta classe você DEVE substituir Strings POR OBJETOS:

### ❌ Variáveis que NÃO podem ser String

Se você colocou algo assim:

* pessoa: String
* cuidador: String
* contatos: String
* idoso: String

Tudo errado.

### ✔ Variáveis que PRECISAM virar OBJETO

Aqui vai o correto:

```
- pessoaQueDisparou: Pessoa   (ou Idoso)
- cuidadorResponsavel: Cuidador
- contatosNotificados: List<ContatoFamiliar>
```
